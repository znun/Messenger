<p>
<h1>Requirements</h1>
<ul>
  <li>Xcode version 11.2.1+</li>
  <li>Swift 5</li>
  <li>iPhone 8 or higher</li>
  <li>iOS 13.0+</li>
</ul> 
</p>

<p>
<h1>Functionality</h1>
<ul>
<li>Real-time chat</li>
<li>Sending text messages</li>
<li>Unlimited length of text messages</li>
<li>Sending image messages</li>
<li>Sending video messages</li>
<li>Sending audio messages</li>
<li>Sending users current location</li>
<li>Delete messages</li>
<li>Change password</li>
<li>Change profile image</li>
<li>Change profile username</li>
<li>Set user status</li>
</ul>
</p>

<p>
<h1>How to install?</h1>
<ol>
<li>Install CocoaPods</li>
<li>Open Terminal and run pod install directly in /Messenger folder.</li>
<li>In order for Firebase to work, create a new project for your application.</li>
<li>Download GoogleService-Info.plist from your newly created Firebase project and replace it with the old one.</li>
<li>Enable Email/Password authentication method.</li>
</ol>
